# Raport z zadania domowego: Ultimate Tic-Tac-Toe Bot

Projekt stanowi rozwiązanie zadania Ultimate Tic-Tac-Toe na platformie CodinGame, gdzie głównym celem było stworzenie bota zdolnego pokonać wbudowane AI przy użyciu algorytmu Minimax z odcinaniem Alpha-Beta. Logika gry wykorzystuje zaawansowaną heurystykę oraz dynamiczną ewaluację głębokości, nakładając odpowiednie kary za niekorzystne stany w kluczowych etapach rozgrywki.
